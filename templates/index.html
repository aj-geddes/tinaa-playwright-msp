Tinaa Playwright MSP

Welcome to Tinaa Playwright MSP, a Playwright-based web testing server using Model Service Protocol.

About Tinaa Playwright

Tinaa is an expert web testing assistant that can help you test websites for functionality, accessibility, performance, and security issues. Powered by Playwright, Tinaa can navigate websites, interact with elements, take screenshots, and analyze page content.

Key Features

  • Navigate to web pages and interact with UI elements
  • Fill in forms and submit data
  • Take screenshots of pages and elements
  • Run accessibility checks and suggest improvements
  • Test responsive design across different screen sizes
  • Execute predefined test patterns for common website features
  • Apply testing heuristics to find potential issues
  • Securely handle credentials for authenticated testing

Getting Started

To interact with Tinaa, you can use the Model Service Protocol API. Here's an example:


    curl -X POST http://localhost:8000/api/mcp \
    -H "Content-Type: application/json" \
    -d '{
        "function": "navigate_to_url",
        "parameters": {
            "url": "https://example.com"
        }
    }'
